# Challenge-ELOQUENT---Cultural-Robustness-Diversity

Pour démarrer l'interface graphique de l'application, ouvrez votre terminal et exécutez la commande suivante :

```bash
streamlit run app.py
```

Si vous n'avez pas la bibliothèque streamlit, vous devez l'installer avec la commande 
```bash
pip install -r requirements.txt
```
